This installation page must be run locally to communicate with your SyftBox instance (which runs on localhost).

To install the CPU Tracker Member app:

  1. Download this page by clicking the button below
  2. Save the file to your computer
  3. Open the downloaded file in your browser
  4. Make sure your SyftBox is running (accessible at localhost:8080)
  5. Click the Install button on the locally opened page
